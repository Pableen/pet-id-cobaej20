# pet-id-cobaej20
El proyecto PET-ID del COBAEJ Plantel 20 consiste en la creación de placas inteligentes para mascotas elaboradas con tecnología NFC (Near Field Communication). Cada placa contiene un chip que, al ser escaneado con un teléfono inteligente, muestra información esencial del animal, como nombre, contacto del dueño y datos para su pronta localización.
